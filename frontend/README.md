
  # EdTech Mobile App Development

  This is a code bundle for EdTech Mobile App Development. The original project is available at https://www.figma.com/design/DthIQowt1CIr2ufQcTfgTb/EdTech-Mobile-App-Development.

  ## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Create a `.env` (or `.env.local`) file in `frontend/` if you need to point the UI at a different backend origin:

```
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

It defaults to `http://localhost:8000/api/v1` when not set.
